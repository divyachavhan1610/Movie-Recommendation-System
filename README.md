🎬 AI Movie Recommendation System
An AI-powered Movie Recommendation System built using Python, Machine Learning, NLP, Streamlit, and TMDB API.

📌 Project Overview
This project recommends movies similar to the one selected by the user using a Content-Based Recommendation System.

The recommendation is based on:

Genres
Keywords
Cast
Director
Movie Overview
The application also displays:

🎬 Movie Poster
⭐ IMDb Rating
📅 Release Date
🎭 Genres
▶️ Movie Trailer

▶️ How to Run
Clone the repository

Install requirements

pip install -r requirements.txt

Create config.py
API_KEY="YOUR_TMDB_API_KEY"

Run
streamlit run app.py
